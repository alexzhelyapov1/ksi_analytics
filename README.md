# ksi_analytics
Статистика распределения бюджета Конкурса Студенческих Инициатив МФТИ по годам и направлениям

# Navigation
```Row documents``` - перечень проектов-победителей КСИ по годам в формате pdf\
```Recognized documents``` - распознанные документы в формате docx\
```src``` - исходники скрипта на питоне, для генерации новых таблиц

# Usage
```python3 main.py``` - запускскрипта.
После выполнения создаст файл формата xlsx, распределение бюджета КСИ по кластерам в зависимости от года (в процентах от общего бюджета конкурса)
